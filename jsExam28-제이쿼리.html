이벤트 효과 메소드
show() : 선택한 요소를 보여준다.
hide() : 선택한 요소를 숨긴다.
toggle() : 선택한 요소를 보여주고 숨기는 것을 반복한다.
fadeIn() : 선택한 요소를 서서히 보여준다.
fadeOut() : 선택한 요소를 서서히 숨긴다.
fadeToggle() : 선택한 요소를 서서히 보여주고 숨기는 것을 반복한다.
slideDown() : 선택한 요소를 아래로 서서히 보여준다.
slideUp() : 선택한 요소를 위로 서서히 숨긴다.
slideToggle() : 선택한 요소를 위/아래로 서서히 보여주고 숨기는 것을 반복한다.
